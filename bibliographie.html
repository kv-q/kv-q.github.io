Sources et explications

  • W3Schools (Navigation, CSS, Scroll)
    W3Schools a été utilisé pour comprendre et implémenter un menu de navigation ainsi qu'un bouton de retour en haut de page. Source, Source, Source
  • Stack Overflow (Retour en haut CSS/HTML)
    Stack Overflow a aidé à trouver une solution pour créer un bouton de retour en haut de page uniquement en CSS et HTML. Source
  • Poetry Foundation (Sonnet 18)
    Cette source a fourni le texte original du poème "Sonnet 18" de Shakespeare utilisé dans l'analyse. Source
  • Byron's Muse (Analyse et image)
    Ce blog a été utilisé pour obtenir une analyse du poème ainsi qu'une image illustrant le thème du sonnet. Source
  • Mozilla Developer Network (Meta Viewport)
    Utilisé pour apprendre comment configurer la balise meta viewport afin d'améliorer l'affichage sur mobile. Source
  • W3Schools (HTML de base)
    Référence pour les bases du HTML et de la structuration des pages web. Source
  • YouTube (Développement Web Moderne)
    Une vidéo informative sur le développement web moderne et minimaliste. Source
  • CodeMentor (Minimalisme Web)
    Un article expliquant les principes du développement web minimaliste et HTML simplifié. Source
